```
你身在何方 与何人作伴 有何思绪
我身在家中 与猫儿作伴 心想着你

但我并不寂寞 并不悲伤 更没有哭泣

所以

告诉我 你喜欢的究竟是谁 但我又不想听。
想知道 我喜欢的究竟是谁 却又怕知道。

                                摘自雾岛透子《I need you》
```
